12-month roadmap workflow

Theory leads to practice, practice creates evidence and evidence receives review before the next stage.

  1. 1Days 1–30

    Baseline and structure

    Skills baseline, GitHub profile, study calendar, lab diagram and threat model.

  2. 2Days 31–60

    Hands-on foundations

    Linux and networking practice, 10–15 labs, agent hardening and first repository.

  3. 3Days 61–90

    Applied evidence

    Log analysis, incident report, second repository and mentor review.

  4. 4Months 4–6

    Cyber foundations

    Identity, endpoints, network defence, CCST and SC-900-aligned learning.

  5. 5Months 7–12

    Operations and transition

    Portfolio maturity, credential readiness, ADF and degree planning.
