## Standard workflow

1. Read the learner-facing assignment specification.
2. Confirm the task and deadline with the mentor.
3. Create an assignment branch.
4. Complete the provided deliverable templates.
5. Record evidence and sources.
6. Open a pull request linked to the tracking issue.
7. Resolve mentor feedback before merge.
