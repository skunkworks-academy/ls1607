## Purpose

Use the external [GitHub Achievements Academy](https://github.com/Puliczek/github-achievements-academy) resource as a short guided GitHub learning activity. The objective is to practise legitimate GitHub workflows and record evidence of what was learned.

The external project describes itself as a short activity for learning more about GitHub and earning profile achievements. Achievement availability and award criteria are controlled by GitHub and may change, so completion is assessed on workflow competence and evidence—not on a guaranteed badge count.

## Start here

1. Read the [external repository README](https://github.com/Puliczek/github-achievements-academy).
2. Open the learning site referenced by that repository.
3. Complete only safe, legitimate and relevant activities.
4. Update [`reflection.md`](reflection.md).
5. Record evidence in [`evidence/README.md`](evidence/README.md).
6. Open a pull request linked to issue [#15](https://github.com/skunkworks-academy/ls1607/issues/15).

## Required deliverables

- [ ] Completed learning activity
- [ ] Short written reflection
- [ ] Evidence register containing relevant public links
- [ ] List of GitHub features practised
- [ ] Mentor-reviewed pull request

## Integrity requirements

- Do not create spam issues, comments, repositories or pull requests merely to manipulate profile achievements.
- Do not tag or involve unrelated users without consent.
- Do not expose personal data, credentials, tokens, private repository content or confidential information.
- Follow GitHub's applicable terms, acceptable-use rules and community standards.
- Use meaningful repository activity that demonstrates actual learning.

## Definition of done

- [ ] Reflection explains what was learned.
- [ ] Evidence links are valid and appropriate for public review.
- [ ] No sensitive information is present.
- [ ] Repository validation passes.
- [ ] Mentor feedback is resolved.
- [ ] Final pull request includes `Closes #15` and is approved and merged.
